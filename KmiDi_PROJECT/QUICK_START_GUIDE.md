This is a super simple guide to get you making music right away. For more details, see the full [USER_MANUAL.md](USER_MANUAL.md).

## üöÄ Fastest Way to Start

### Option 1: Web App (Easiest - No Installation)
1. Open your web browser
2. Go to: `https://kmidi.app`
3. Type how you feel in the box
4. Click "Generate Music"
5. Done! üéµ

### Option 2: Desktop App (Most Features)
1. Download KmiDi from the website
2. Install it (drag to Applications on Mac, or run installer on Windows)
3. Open KmiDi
4. Type your feelings
5. Click "Generate Music"

### Option 3: Mobile App (On the Go)
1. Search "KmiDi" in App Store (iPhone) or Play Store (Android)
2. Install the app
3. Open it
4. Tap the microphone or type your feelings
5. Tap "Generate"

---

## üìù Your First Song in 3 Steps

### Step 1: Describe Your Feeling
Type something like:
- "I'm feeling happy and energetic"
- "This is about missing someone"
- "I want something calm and peaceful"

### Step 2: Click "Generate Music"
Wait 10-30 seconds. That's it!

### Step 3: Listen and Save
- Click the play button to hear it
- Click "Save" if you like it
- Click "Generate" again if you want to try something different

You don't need to use these, but they help:

1. **Motivation** (Left to Right)
   - Left = Calm
   - Right = Energetic

2. **Chaos** (Left to Right)
   - Left = Predictable
   - Right = Wild

3. **Vulnerability** (Left to Right)
   - Left = Guarded
   - Right = Emotional

**Tip:** Start with them in the middle, then move them around to see what happens!

## üéØ Common First-Time Questions

**Q: Do I need to know music?**  
A: No! Just describe your feelings.

**Q: Is it free?**  
A: Check the website - there's usually a free version to try.

**Q: Can I use this in Logic Pro/GarageBand?**  
A: Yes! Install the plugins (see full manual).

**Q: How do I save my music?**  
A: Click "Save" button, or "Export" to save as a file.

**Q: Can I edit the music?**  
A: Yes - regenerate with different settings, or export to edit in music software.
