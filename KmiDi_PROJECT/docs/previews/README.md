# UI Previews

This directory contains preview screenshots for KmiDi documentation.

## Required Screenshots

1. **preview_main_window.png** - Standalone app main window showing timeline and emotion inspector panel
2. **preview_emotion_inspector.png** - Emotion inspector panel with emotion radar/controls
3. **preview_plugin_editor.png** - JUCE plugin editor (standalone or in host mock)

## Generation Instructions

To generate these screenshots:

1. Launch the KmiDi standalone app
2. Capture the main window showing the timeline and emotion inspector
3. Open the emotion inspector panel and capture the emotion radar
4. Launch the JUCE plugin (standalone or in a DAW host) and capture the editor

Save screenshots as PNG files with the names listed above.
